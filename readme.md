## 赛制简介

赛制共分为五个阶段：
提名阶段-预选赛阶段-小组赛阶段-淘汰赛阶段-最终决赛（可视为淘汰赛最终阶段）

### 提名阶段

用户在提名阶段时可在“提名”页中通过搜索进行提名

### 预选赛阶段

预选赛阶段时，用户可在“预选赛”页中进行投票，每人最多投出20票
阶段结束时，将统计前32名晋级小组赛

### 小组赛阶段

小组赛将根据预选赛的投票结果排名进行蛇形分组，共分为8组，每组4名角色
每组中，进行两两对决，胜积3分，平积1分，负积0分，每组前两名晋级16强

### 淘汰赛阶段

根据小组赛的结果进行分区，如 A1-H2,B1-G2,···，进行两两对决，共3轮（16进8，8进4，4进2）

### 最终决赛

本阶段包含冠军赛与季军赛，同样为两两对决
